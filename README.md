# Pacman-like Game

ブラウザで動作する、シンプルな **パックマン風 2D ゲーム**です。  
HTML / CSS / JavaScript のみで実装しており、ビルド不要で動作します。

---

## 🎮 操作方法

- **↑ ↓ ← →（矢印キー）**  
  プレイヤーの移動

- **R キー**  
  ゲームのリスタート（ゲームオーバー時 / クリア後）

---

## 📜 ルール・仕様

### スコア
- ドット取得：**+10**
- パワーアイテム取得：**+50**
- パワー状態中に敵を倒す：**+200**

### ライフ
- 初期ライフ：**3**
- 敵に接触すると **ライフ -1**
- ライフが 0 になると **ゲームオーバー**

### パワー状態
- パワーアイテム取得後、一定時間 **敵を倒せる状態**になります
- パワー状態中は敵が点滅表示されます
- パワー状態が終了すると通常状態に戻ります

### 勝利条件
- ステージ内のドットをすべて取得すると **クリア**
